# E-Commerce Order Management Database System
# WEEK-1 Understanding the Business Problem
## Project Overview

This repository contains the Week 1 requirement analysis for the E-Commerce Order Management Database System. The project focuses on understanding the business requirements before designing the database.

## Contents

* Requirement Analysis Report
* Business Requirement Document
* Stakeholder Analysis
* Functional Requirements
* Non-Functional Requirements
* Project Scope
* Assumptions and Constraints
* Software Requirement Specification (SRS)

## Project Objective

To design a centralized database system that efficiently manages customers, products, suppliers, orders, payments, shipments, and reviews while reducing redundancy and improving operational efficiency.
